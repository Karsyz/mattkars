Full Stack Developer

Sofware Engineering is my latest endeavour. I've been writing code for a good long while, but mostly for robotics. With the latest machine control systems implementing web-based human-machine-interfaces, adding the web-dev to my skill set was a natural progression.

3D Mechanical Design

I really love designing things! My first experience with computer aided drafting and design (CAD) was in early highschool (1995). I was hooked on it right from then. Nowadays I like to design everything from 3D printables to functional machines and just about everything in between.

Industrial Robotics

I got into industrial robotics setup and programming in 2008, when I was invited to participate in a shutdown retool for a major automotive assembly plant. Again, I wa hooked from then on. I don't do this type of work full time anymore, but I still love it.
